# 🐰 BunnyEra — 兔子字体套件

兔子字体套件是 BunnyEra 官方字体与视觉资源库，提供品牌字体、图标、排版规范等资源。

---

## 🚀 功能模块

- 官方品牌字体（TTF/OTF）
- 字体使用规范
- Logo 字体与图标资源
- 视觉系统基础规范

---

## 📦 使用方法

1. 下载字体文件（/fonts）
2. 安装到系统或项目中
3. 按照视觉规范文档使用
